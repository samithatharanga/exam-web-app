## 📝 Project Overview
This repository contains a **Machine Learning Web Application** developed collaboratively by **Group C** as a practical assessment for the **3rd Year, 1st Semester Examination** at the **Nāgānanda International Institute for Buddhist Studies (NIIBS)**.

Our team built a custom Artificial Intelligence model using **Transfer Learning** on the **MobileNetV2** architecture to accurately classify hand gestures for Rock, Paper, and Scissors. The backend model is seamlessly integrated into a highly interactive, real-time web interface using **Streamlit**.

## ✨ Key Features & Experience
* **Computer Vision:** Fine-tuned MobileNetV2 model specifically trained for hand gesture recognition.
* **Real-time Predictions:** Users can upload images directly to the web app and receive instant classifications.
* **Streamlit Deployment:** A responsive and interactive UI hosted on Streamlit Community Cloud.
* **Team Collaboration:** Successfully managed an end-to-end ML pipeline as a group, from dataset preparation to cloud deployment.
